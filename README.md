# Divya Makeup Artist

Static website for Divya Makeup Artist, Hyderabad.

## Project structure

- `index.html` — main page
- `assets/css/style.css` — site styles
- `assets/js/script.js` — site interactions
- `assets/images/gallery/` — portfolio images

## Local development

1. Open the project folder in VS Code.
2. Run a local server:
   - `python3 -m http.server 8000`
   - open `http://localhost:8000`
3. Or use Live Server extension.

## Git setup

```bash
git init
git add .
git commit -m "Initial project setup"
git branch -M main
# replace with your GitHub repository URL
git remote add origin https://github.com/your-username/divya-bridal.git
git push -u origin main
```
